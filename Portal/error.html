Oh Snap ... an error page

This page gets displayed when one of the following happens, in order of likelihood:
  • A test type requested from the main portal does not exist
  • Grafana tries to display an annotation that no longer exists*
  • An incorrect URL was manually entered
  • The web server encountered an issue
* HTML test results and Grafana annotations are saved for 7 days, but Graphite saves data for 13 months
