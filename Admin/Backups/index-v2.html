GS Advanced Services - Site Analysis Portal

Welcome to the GS Advanced Services Site Analysis Portal. The purpose of this site is to display the results of synthetic testing driven by sitespeed.io. The objective of the testing is to provide an easy and fast way to test up to five different URLs for a site in order to gather information that can serve as the basis for doing a site analysis. The data can be used in a standalone manner or compliment existing mPulse data.

In addition to testing a single domain, a competitive analysis can be done for a specific industry (e.g., eCommerce, airlines, financial). A competive analysis targets up to seven different site URLs. Both Single Domain and Competitive Analysis testing is easy to configure and can be set up quickly.

Some highlights include:

  • Site and Page Metrics
  • Site Construction
  • 1st & 3rd Party Domain Information
  • Largest JS, CSS, and Images
  • Slowest Responses
  • Core Web Vitals
  • Waterfall
  • Filmstrip
  • Video
  • and much much more ...
Results can be reviewed in Grafana, which provides time series dashboards. Or, view the auto-generated Sitespeed.io HTML-based reports, which provide extremely rich information along with perscriptive advice on how to improve site performance. Historical data in the database is saved for 13 months.
