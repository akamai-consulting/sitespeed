GS Advanced Services - Site Analysis Portal

Welcome to the GS Advanced Services Site Analysis Portal. The purpose of this site is to share the results of synthetic testing driven by sitespeed.io. The objective of the testing is to provide an easy and fast way to test different URLs for a site to gather information that can serve as the basis for doing an in-depth analysis. The data can be used in a standalone manner or compliment existing mPulse data. Site testing is done from Linodes servers that are located in each of Linode's regional data centers. To get an overview of the Site Analysis Portal and learn more about the architecture and underlying software components, refer to Sitespeed Overview.

Testing can be done for either a single domain with multiple pages or for URLs across multiple domains (e.g., eCommerce, airlines, financial), which would be the basis for a competitive analysis.

Highlights include:

  • Site and Page Metrics
  • Site Construction
  • 1st & 3rd Party Domain Information
  • Largest JS, CSS, and Images
  • Slowest Responses
  • Core Web Vitals
  • Waterfall
  • Filmstrip
  • Video
  • PageSpeed Insights
  • Lighthouse
  • Chrome User Experience
and much much more ...
